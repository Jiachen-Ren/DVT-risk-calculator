DVT web calculator package
==========================

This folder was automatically generated by the DVT machine-learning pipeline.

Files:
- app.py
- requirements.txt
- final_model_bundle.joblib
- final_model.pkl
- final_preprocessor.pkl
- selected_variables.json
- model_threshold.json

How to run locally on Windows:
1. Open CMD.
2. Enter this folder:
   cd /d "E:\python\工程\学习\ICU-DVT\DVT_results\DVT_web_calculator_streamlit_package"
3. Install requirements:
   python -m pip install -r requirements.txt
4. Start the calculator:
   python -m streamlit run app.py

Then open:
http://localhost:8501

Important:
- The calculator uses the final ElasticNet model.
- Laboratory values must use the same units as the modeling dataset.
- The calculator is a research prototype and cannot replace clinical judgment or lower-extremity ultrasound.
